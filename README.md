## Sistema de Cadastro de Funcionários com Python e MySQL

## Sobre o projeto

Este projeto é um sistema de cadastro de funcionários em modo console, desenvolvido em Python, integrado a um banco de dados MySQL.
Ele permite realizar operações completas de CRUD (Criar, Ler, Atualizar e Excluir) em uma tabela de funcionários.

O projeto possui duas versões do código:

Uma versão inicial (main.py)

Uma versão revisada e aprimorada (reviewed.py), com boas práticas e maior segurança

## Tecnologias utilizadas

Python

MySQL

MySQL Connector (mysql-connector-python)

## Funcionalidades

Inserir funcionários no banco de dados

Listar todos os funcionários cadastrados

Atualizar dados de um funcionário pelo ID

Excluir funcionário pelo ID

Menu interativo no terminal

## Objetivo

Projeto desenvolvido para praticar integração entre Python e MySQL, reforçando conceitos como:

Conexão com banco de dados

Execução de comandos SQL via Python

Estruturação de funções

Boas práticas de segurança (queries parametrizadas)

Lógica de menus interativos

## Observações

O banco de dados utilizado chama-se projeto

A tabela principal é funcionario

A versão reviewed.py utiliza queries parametrizadas, evitando SQL Injection

O sistema é executado via terminal (console)
